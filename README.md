#  Contenido del Laboratorio
Este laboratorio incluye ejemplos prácticos de desarrollo web con HTML5 y CSS3, aplicados en distintos archivos:  

-  index.html  Página principal con información institucional.  
-  formulario.html  Formulario básico.  
-  tabla.html  Tabla de estudiantes.  
-  login.html  Ejemplo de login estilizado.  
-  card.html  Tarjeta de producto destacada.  
-  Archivos CSS para los diferentes módulos.  

#  Objetivos del Laboratorio
1. Practicar la estructura básica de un documento HTML5.  
2. Aplicar estilos CSS personalizados.  
3. Utilizar etiquetas semánticas correctas.  
4. Integrar imágenes, tablas, formularios y enlaces.  
5. Crear un proyecto organizado y navegable.  

#  Pasos 
- Creación del repositorio en GitHub.  
- Desarrollo de los archivos HTML.  
- Creación de hojas de estilo CSS.
